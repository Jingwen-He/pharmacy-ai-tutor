# Pharmacy AI Tutor

AI-powered study companion for pharmacy students. Upload your course PDFs and get:

- **Q&A Mode** — Ask questions about your materials with cited answers
- **Quiz Mode** — Practice with auto-generated MCQs, true/false, short answer, and case-based questions

Built with Streamlit, Claude API, LangChain, and ChromaDB.
